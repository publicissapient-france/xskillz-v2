# xskillz-v2 - Back

## Install modules

```shell
$ make install
```

## Test the application

```shell
$ make test
```

## Build the Docker image

```shell
$ make build-image
```

## Push the Docker image

```shell
$ make push-image
```

## Run the application

```shell
$ docker run -d -p 8080:8080 xebiafrance/xskillz-back 
```
